# Prompt: Polish Existing UI

```text
Review and polish the existing UI in [FILE_OR_ROUTE].

Goals:
- Make the UI feel like a real product screen.
- Remove generic AI-looking patterns.
- Improve hierarchy, spacing, alignment, density, and interaction states.
- Use the repo's design tokens.
- Preserve functionality.

Check:
- layout rhythm
- typography hierarchy
- surface/elevation usage
- border radius consistency
- loading/empty/error states
- keyboard accessibility
- focus visible
- responsive behavior
- dark mode

Do not introduce a new design system. Work within the current codebase.

Make the smallest useful changes, then run relevant checks.
```
