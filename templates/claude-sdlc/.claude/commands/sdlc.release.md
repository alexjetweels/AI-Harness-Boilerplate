Create or update `docs/sdlc/current/11-release.md`.

Include:

- Release summary.
- Requirements completed.
- Verification status.
- Security and review status.
- Migration or rollback notes.
- Operational risks.
- Final decision: releasable, blocked, or needs approval.

Do not claim the change is releasable if review, test, or security artifacts are missing.
